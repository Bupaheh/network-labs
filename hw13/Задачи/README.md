### 3

Время опроса одного узла: $2d_{опрос} + \frac{Q}{R}$

Время для опроса всех узлов: $N(2d_{опрос} + \frac{Q}{R})$

Тогда пропускная способность для одного конкретного узла: $\frac{Q}{N(2d_{опрос} + \frac{Q}{R})}$
